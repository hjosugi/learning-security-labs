# OWASP Input Validation Lab

Local-only defensive coding lab for input validation and safe query building.

## Run

```bash
python3 projects/owasp-input-validation-lab/lab.py
python3 projects/owasp-input-validation-lab/test_lab.py
```

## What To Learn

- why string-built queries are dangerous
- allow-list validation
- parameterized query shape
- writing a regression test for the fix

This lab does not target any real system.
